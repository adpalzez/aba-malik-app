## 📖 نبذة عن المشروع | About The Project
تطبيق ويب متكامل يجمع بين الجوانب الروحية، الترفيهية، والمهنية. تم بناؤه باستخدام تقنيات الويب الحديثة ليعمل كـ **تطبيق ويب تقدمي (PWA)** يوفر تجربة مستخدم سلسة على كافة الأجهزة.

An integrated web application that combines spiritual, entertainment, and professional aspects. Built with modern web technologies as a **Progressive Web App (PWA)**.

---

## 🚀 المميزات الرئيسية | Main Features

### 🕌 ركن المسجد (The Mosque Corner)
- **مواقيت الصلاة:** حساب دقيق بناءً على الموقع الجغرافي (API).
- **بوصلة القبلة:** تحديد اتجاه القبلة برمجياً.
- **التقويم الهجري:** عرض التاريخ الهجري والميلادي بشكل متزامن.
- *Prayer timings, Qibla compass, and Hijri calendar integration.*

### ⚔️ مملكة صلاح الدين (Kingdom of Saladin)
- **لعبة استراتيجية:** إدارة الموارد (ذهب، خشب)، بناء القلاع، وتجنيد الجيوش.
- **نظام المعارك:** تعرض المملكة لهجمات عشوائية تتطلب دفاعاً استراتيجياً.
- **حفظ التقدم:** استخدام `LocalStorage` لضمان عدم ضياع تقدم القائد.
- *Strategy game featuring resource management, castle building, and battle systems.*

### 📿 موسوعة الأذكار (Azkar Encyclopedia)
- واجهة مريحة لقراءة الأذكار والأدعية اليومية بتصميم متناسق.

---

## 🛠️ التقنيات المستخدمة | Tech Stack
* **Frontend:** HTML5, Tailwind CSS, JavaScript (ES6+).
* **Animations:** Animate.css & FontAwesome.
* **Deployment:** Vercel / GitHub Pages.
* **Icons:** Icons8.
